# 🎬 BookMyShow Clone

A full-stack movie ticket booking application built with React.js and Spring Boot.

## 🌐 Live Demo
- **Frontend:** https://bookmyshow-ui-by-jaivy-roy.vercel.app
- **Backend API:** https://bookmyshow.ddns.net/api

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Axios
- CSS3

**Backend:**
- Spring Boot (Java 17)
- MySQL (AWS RDS)
- Hibernate/JPA

**DevOps:**
- AWS EC2 (Backend Hosting)
- AWS RDS (Database)
- Nginx (Reverse Proxy + SSL)
- Vercel (Frontend Hosting)
- Let's Encrypt (SSL Certificate)

## ✨ Features
- Browse movies with posters and ratings
- View theaters by city
- Select shows and seats
- User authentication
- Ticket booking system

## 🚀 Setup & Installation

### Backend
```bash
git clone https://github.com/CODEWITH-JAIVY/bookmyshow.git
cd bookmyshow
mvn clean package -DskipTests
java -jar target/BMS-0.0.1-SNAPSHOT.jar
```

### Frontend
```bash
git clone https://github.com/CODEWITH-JAIVY/bookmyshow-ui.git
cd bookmyshow-ui
npm install
npm start
```
